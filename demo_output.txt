# Cargo-GO Image Description Tool - Demo Output

## Build the application
$ go build -o cargo-go

## Show help message
$ ./cargo-go
Usage: cargo-go <image-file>
Example: cargo-go image.jpg

## Describe a sample image
$ ./cargo-go sample.png
Image Description:
  Filename: sample.png
  Format: png
  Dimensions: 800x600 pixels
  Orientation: Landscape
  Aspect Ratio: 1.33:1
  File Size: 2.7 KB
